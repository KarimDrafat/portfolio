Project description

In this project, I conducted a comprehensive analysis of a marketing A/B testing dataset aimed at evaluating the effectiveness of advertising campaigns. I utilized various statistical methods to preprocess the data, perform exploratory data analysis (EDA), and visualize the results.

Full code can be found here

The key steps

  • Data Preprocessing: Cleaning and organizing the dataset to ensure accuracy in analysis.
  • Exploratory Data Analysis (EDA): Uncovering patterns and insights regarding user behavior across different test groups.
  • Statistical Testing: Implementing A/B testing techniques, including:
    • Binomial tests to assess conversion rates
    • Chi-squared tests to evaluate categorical variables
    • Shapiro-Wilk tests for normality
    • G-tests for goodness of fit
    • T-tests to compare means between group

Assessing Campaign Effectiveness

Through this analysis, I aimed to determine whether the advertising campaign was successful and quantify the impact of the ads on user conversion. The findings contribute valuable insights for marketing strategies, enabling companies to make data-driven decisions that enhance campaign effectiveness and optimize resource allocation.
